Is the site's code open source?

Yes. The site is published under a GPL v3 license. The site is built with Jekyll, Bootstrap, Flickity.

What are artifact's references?

Each artifact has a unique reference to identify it. The categorization scheme is loosely based on Dewey Decimal System. The first three algarisms represent the categorization and the last four the artifact's number within that categorization.
